Listas

Listas não ordenadas

Listas não ordenadas não possuem indicação de ordem como numeros ou letras
elas inicial e finalizam com a tag "ul" tendo em sua parte interna tags "li"
que são as linhas da lista da tabela.

Dica de atalho coloque a tag pai ex: ul>li*numero de linhas da lista
que vc quer que o vcscode monta para vc.

Ex:

  • linhas da lista
  • linhas da lista
  • linhas da lista
  • linhas da lista
  • linhas da lista

Lista Ordenada

Listas ordenadas como o proprio nome diz são listas que
possuem alguma indicação de ordem como numeros, letras, ... para indicar
o posicionamento de cada item ele começa e termina com a tag "ol" e as
as linhas tema mesma tag que a lista não ordenada as tags "li".

Ex:

  1. linhas da lista
  2. linhas da lista
  3. linhas da lista
  4. linhas da lista
  5. linhas da lista

Lista com descrição dos elementos

A tag "dl" nos permite criar listas com descrição como se fosse
um titulo para cada linha onde a tag "dt" é como se fosso titulo e a
tag "dd" é a descrição

EX:

"dt" tema/ titulo
"dd" descrição
café
Pingado R$ 2,50
café
