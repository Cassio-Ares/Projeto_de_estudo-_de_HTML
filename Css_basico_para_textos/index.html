CSS básico para estilização de textos

OBS:Normalmente a estilização é feita em uma outra pagina
com nomes como "index.css" ou "style.css"... porém para melhor visualização do codigo a estilização aqui será feita inline

color

Propriedade CSS que altera a cor do texto

Color: blue/ #0000ff/ rgb(0, 0, 255)/
As cores no CSS podem ser colocadas por nome: blue; pela sua forma Hex: #0000ff
ou RGB: rgb(0, 0, 255)

Alinhamento de Texto

Text-align

Propriedade usada para ajustar o alinhamento horizontal do texto.

Texto alinhado ao centro text-align: center

text-align: left, center, rigth
text-align: justify; faz todas as linhas se alinharem para texto ficar uniforme

Direction

Propriedade que defina onde o texto aparecerá na pagina "ltr" normal a esquerda e "rtl" a direita

Texto escrito ao contrario

direction: rtl;

Unicode-bidi

Propriedade que define a direção que as letras irão seguir da esquerda para direita "normal"
ou da direita para a esquerda "bidi-override"

Texto escrito ao contrario

OBS:Para ter o efeito de inversão é preciso unir dois comandos direction: rtl; e o unicode-bidi: bidi-override;
