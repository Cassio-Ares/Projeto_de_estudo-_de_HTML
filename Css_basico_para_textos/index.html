Direction

Propriedade que defina onde o texto aparecerá na pagina "ltr" normal a esquerda e "rtl" a direita

Texto escrito ao contrario

direction: rtl;

Unicode-bidi

Propriedade que define a direção que as letras irão seguir da esquerda para direita "normal"
ou da direita para a esquerda "bidi-override"

Texto escrito ao contrario

OBS:Para ter o efeito de inversão é preciso unir dois comandos direction: rtl; e o unicode-bidi: bidi-override;

Writing-mode

Propriedade para definir a direção das linhas horizontal ou vertical

Texto com writing-mode: horizontal-tb

Vertical-align

Propriedade define o alinhamento vertical de uma elemento.

Texto normal e a imagem esta com a propriedade vertical-align: super

A propriedade "vertical-align" sup, super, botton, ...diversos comandos para alinhamento.

Propriedade permite adicionar linha(s) para decorar o texto.

Texto com uma linha na parte da baixo para dar destaque

A propriedade permite colocar linha(s) na parte superior(overline), sobre o texto(overline), undeline como no exemplo...

Text-decoration-color

Propriedade permite colocar cor nas linhas de decoração.

Texto com a a propridade text-decoration: overline underline;

Text-decoration-style
