Tags para Tabelas no HTML

Table

Tag principal para criar a tabela ela engloba toda a tabela
<.table> todos os elementos da tabela <./table>

Caption

Tag para criar a leganda da tabela, fica logo abaixo de "table"

Thead

Tag para agrupar os conteúdo de cabeçalho serve para organizar melhor
dividindo a tabela em Header, body e footer se nescessario

Tbody

Tag que define parte do corpo da tabela

Tfoot

Tag que define o footer da tabela

Tr

Tag que define uma linha em uma tabela

Th

Tag que define as celulas do Theader(cabeçalho)são colocadas
dentro da tag "tr" a linha do cabechalho

Td

Tag que constroi as celulas de dados usada dentro das linhas "tr"
dentro do "tbody"

O tfooter não tem tags especificas, ele deve ser construido com o mesmo modelo de tags que o tbody

Tags para fazer estilização

Colgroup

Tag que especifica um grupo de colunas para estilização

Col

Tag onde é colocada as especificações para as colunas

<.table>
<.colgroup>
<.col span="2" style="background-color:Pink">
<.col style="background-color:yellow">
<./colgroup>

<.tr>
<.th>HTML
<.th>CSS
<.th>JS
<./tr>

OBS:cada <.col> irá estilizar uma coluna

para estilizar 2 ou mais colunas em seguidas usamos
"span= e o numero de colunas" nume

Exemplo de tabela completa

<.table>

<.thead>
<.tr>
<.th>cabechalho 1
<.th> cabechalho 2
<./tr>
<./thead>

<.tbody>
<.tr>
<.td>celula 1
<.td>celula 2
<./tr>
<./tbody>

<.tfoot>
<.tr>
<.td>celula 1
<.td>celula 2
<./tr>
<./tfoot>

<./table>
